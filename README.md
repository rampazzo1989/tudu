## Some Technologies Used üõ†Ô∏è

Tud√∫ is built using the following technologies:

- **Recoil.js for State Management:** Tud√∫ utilizes Recoil.js to manage state in a simple and scalable way, making it easy to handle complex state logic.

## Getting Started üöÄ

Follow these simple steps to get started with Tud√∫:

1. **Clone the Repository:**
```
git clone https://github.com/your-username/tudu.git
```
2. **Navigate to the Tud√∫ Directory:**
```

```
3. **Install Dependencies:**

- For Android (using npm):
  ```
  npm install
  ```

  - or with Yarn:
  ```
  yarn install
  ```

- For iOS (using npm) _(iOS still not tested)_:
  ```
  npm install
  cd ios && pod install && cd ..
  ```

  - or with Yarn:
  ```
  yarn install
  cd ios && pod install && cd ..
  ```

4. **Run the Application:**

- For Android:
  ```
  npm run android
  ```

  - or with Yarn:
  ```
  yarn android
  ```

- For iOS _(iOS still not tested)_:
  ```
  npm run ios
  ```

  - or with Yarn:
  ```
  yarn ios
  ```

5. **Explore and Contribute:**
Explore the source code, try out the app, and if you wish, contribute to make Tud√∫ even better.
